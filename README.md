# xcli

The official Xedix versioning cli tool.

## How to use

1. Download the source

2. Navigate to the folder in the terminal

3. run `cd back`

4. run `python3 xedix.init.py`

- Enter the files you are changing
- Enter commit message
- Make changes

5. Check the changes

- run `cd ..`
- run `cd front`
- open the `index.html`
